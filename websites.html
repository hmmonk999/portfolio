Technical Communication at Iowa State

hannahmonkdot.github.io/TechnicalCommunication/index.html

I built this website for a visual rhetoric class in the Fall of 2019. I sent out a survey to all the technical communication majors at Iowa State. 10 out of 21 students responded. The purpose of the website was to try to define technical communication and how students use it as a major at Iowa State. I created the images using Adobe Illustrator, the charts using Google Charts, and the website using HTML, CSS, and JavaScript.





String Instraments

hmmonk999.github.io/stringInstraments/

With this website I wanted to mess around with SVGs and onkeydown and onkeyup JavaScript events. Each string makes a noise and lights up when a key is pressed and reverts back to its origional state when the key is released.
