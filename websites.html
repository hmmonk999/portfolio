Technical Communication Student Research Website

⧉Data visualization website built with Bootstrap, HTML, CSS, and JavaScript

Deliverables

User Survey
User Survey Results
Visualization of Results

Audience

Undergrad, Graduate, and Faculity in the Iowa State English Department

Website Purpose

Help solidify a definition for technical communication based on different student experiences and interests and further explain those student's experiences.

Work Process
  1. Create and send survey to students majoring in Technical Communication
  2. Analyze the data and decide the best graph type to present the results
  3. Analyze data from Iowa State's Enrollment Statistics Factbook and decide the best way to present the data
  4. Design icons and flat art of students in Adobe Illustrator
  5. Build a website using a Bootstrap template, HTML, CSS, JavaScript, and Google Charts
Drawn Mock Design


Survey Design

Topics: opinions on technical communication, demographics, and future plans

This research project uses qualitative data to increase the understanding of the differnt experience of technical communication studetns. Questions were mostly closed ended with the exception of a few open ended opinion questions. All questions were worded to aviod invoking bias in the user, including not using absolute verbage.
The survey style and questions were tested by users before and updates were made to prevent unclear questions and provide a positive user experience.

Data Analyzation

The data was analyzed in Microsoft Excel and a text analyzer.
