Solar System Design Game

3D game built in Unity3D with C#

Current Design Features

  • Planet Customizer with 20 planet bodies, customizable orbit paths and speed, optional moon addition, and 3 size options
  • Planet edit and planet destroy buttons
  • Planets get destroyed when they get to close to each other
  • Front and top camera views
  • Zoomable Camera
  • Number of current active planets counter
  • Number of total destroyed planets counter

Upcoming Design Features

  • Design mode vs Challenge mode
  • Multiple sun types/levels
  • Score card between levels
  • Error message explaining how a planet got destroyed
  • Sprite animation and sound when a planet is destroyed
  • Loading screen
Press the "Front View" button under Camera View to access the planet menu. Press a plus button under the Planet Controler to add a planet.
