Client Services Intern

Leepfrog Technologies | May 2018 - August 2018

Leepfrog is a tech company that sells web-based software to universites. You may be familiar CAT (short for catalog) because of the Iowa State Course Catalog or the University of Iowa Course Catalog. I worked mostly on client-facing documentation for some of software.


Major Projects

"What To Do Now" Document

This document is meant to inform new clients about their upcoming implementation process. I recreated this document with another intern. We turned a 10 page document into a 3-page document, reorganized it so the main points had a better flow and readabilty, and designed new graphics to get rid of the clipart.


"Now That You Have This Set Up" Document

This document is meant to be similar to the "What to Do Now" document, but given at the end of the implementation process. Another intern and I created it from scratch. We wrote it while working with the implementation department to get all the facts correct.


CLSS Calculator

I gathered information on the implementation process for the CLSS software using Excel. I got rid of the outlines and found the average time between steps to make the timeline caluclator more accurate.

.

Information Architecture on the Internal Website

Leepfrogs internal website is organzied by software (CLSS, CAT, or CIM). I worked with the CLSS team to reorganize the navigation of their part of the internal website. The goal was to make it more similar with the other parts of the website and more accurate with updated information.
